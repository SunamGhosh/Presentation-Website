What is Coding?

Coding can be defined as writing instructions for computers and other hardware. The computer is then able to read the instructions (called “programs”) and do what you have asked it to do. Computer language is different from human language. A human needs to learn “computer language” (commonly known as programming languages) to write programs that the computer can understand. Some “computer languages” include Python, Java, C, C++, and JavaScript. `
